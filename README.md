
# 49.5 Food Delivery
Inspirado no **99 food**, esse é um pequeno front-end que eu fiz para um exercicio do curso do Instituto Evoluir, feito com React + TypeScript, com objetivo de testar minhas habilidades tanto em desing como em lógica de programação e como implementar features de forma certa e futuramente escalável.
### Para rodar:
Primerio, se deve clonar o repositório
```bash
git clone https://github.com/Zumbisinho/49.5-Food-Delivery.git
```
Logo após isso, se deve instalar as dependencias do projeto:
```bash
   npm install
```

Após baixar todas as dependências, é so rodar o projeto
```bash
   npm run dev
```
E entrar na porta criada.
# E se eu não tivar o Node?
Você instala, uai? [Node.js](https://nodejs.org/pt-br) <- baixe e execute tudo certo e é para você ter o npm (Node Package Manager)

# É tão facil assim iniciar um projeto em React?
Com todo o código pronto obvio né, mas é apenas esses passos para desfrutar dessa peça de software abençoada.
